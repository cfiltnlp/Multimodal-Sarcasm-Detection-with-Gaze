This folder contains the codes for extracting various features from different modalities
------------------------------------------------
extract_video_keyframes.py - Extracts keyframes from input videos using Katna Library
extract_key_video_features.py - Extracts the ResNET152 features from selected key frames
audio_feature_extraction.ipynb - Extracts audio features from given audio files
extract_BART_features.ipynb - Extracts BART features from text modality
et_misc.ipynb - Contains different tasks related to eye-tracking experiments like data formatting, IAA calculation etc.
