This folder contains the files used for the project:
	-execute_sarcasm_mustard : Script to train the models for Multimodal sarcasm detection on MUStARD dataset
	-execute_sarcasm_mustard++ : Script to train the models for Multimodal sarcasm detection on MUStARD++ dataset
	-getmustardPP_utext.py : Script to attach gaze features to the other three modalities in a pickle file.
	-execute_autoencoder.py : Script to reduce the dimentionality of text features for gaze prediction using auto encoder.
